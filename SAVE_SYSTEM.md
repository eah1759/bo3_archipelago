# Save System

#### How does it work?

Your progress in a map is saved when either:
- A new round starts
- You end the game via the main menu

This progress is then restored when you enter the map again.

The game ending when everyone dies will cause this progress to be wiped instead.

The intention is to allow you to complete checks on different maps without losing (too much) progress on maps you've already played some of. You can finish half a map at the start of the AP, and clear the rest nearer the end.

#### Checkpoints

You can checkpoint the map via:
- Round Checkpoints (every X rounds)
- EE Checkpoints (up to 3 made when certain conditions are met)
- Spending Checkpoint Tokens (via the Archipelago Shop)

Checkpoints keep your progress even if you die. When you game over, loading into the map will load the last saved checkpoint instead of fresh.

You can choose to clear any checkpoints on a map with Clear Save Data in the main menu. This will also refund any Shop Tokens spent during the map.

## All Maps

As a rule, all maps will save this data:
- Current round number
- Number of remaining zombies in the round
- Doors opened
- Player data
  - Score
  - Loadout, 
    - Weapons including ammo clip + stock
    - Special Equipment / Grenades including number held
    - Specialist Weapon including power
    - Shield
    - Current Gobblegum

Most maps will save and restore power.

## Shadows of Evil

The pack-a-punch ritual will need to be completed each time you restore.

Mementos for already completed rituals will still be able to be picked up, but will not do anything.

Saved and restored:
- Each smashable individually (melee'd with beastmode)

- Each of the 4 rituals individually
- Pack-a-punch room door opened
- Sword locker door opened
- Player data
  - Tier 1 sword acquired
